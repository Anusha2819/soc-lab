Push to GitHub

Step-by-step — create your repo and push all files

  1. Create repo on GitHub.com

    Go to github.com → New Repository → Name it soc-lab → Public → No README (we have one)

  2. Clone or init locally 
    git clone https://github.com/YOUR_USERNAME/soc-lab.git
cd soc-lab
  3. Copy all project files in

    Copy the downloaded soc-lab/ folder contents into your cloned repo folder

  4. Make scripts executable 
    chmod +x scripts/*.sh
  5. Commit and push 
    git add .
git commit -m "Initial commit — Home SOC Lab setup"
git push origin main
[✓] Pushed to GitHub!
  6. Add real screenshots as you complete each phase 
    scrot screenshots/phase2_suricata_running.png
git add screenshots/
git commit -m "Add Phase 2 screenshots"
git push
✓ Your repo will be live at: github.com/YOUR_USERNAME/soc-lab
